Intro

This is a simple tool to check a page if an element exsists or appears and if so, it will send a notification using the OS notification system.

When you first access the extension, the icon is hidden under the "Extensions" button ont the browser toolbar (as highlited in the agissant image)

You can now if you wish, click on the pin icon in order to stick the extension on the toolbar or (left) click the icon in order to access the user intervace (UI) of the extension.

The following list will tell you what each element of the UI does:

  1. "Add/Remove Current URL" button will add the active tab URL into the monitoring list. Clicking it again, will remove the URL if you are on the same page as the URL.
  2. "Attribute field" is where you have to add the element type you want to check for (i.g. class, id, [custom attibute]...).
  3. "Value field" must contain the value of the type/attribute put in Attribute field (i.e. for a div element with class="column", you insert column).
  4. "Add" button will append the entries of Atribute and Value fields into the elements to track list. (Note: If you try to add the same combination of Atribute and Value twice, the entries will be removed from the list.
  5. "Timer field" and "Set button" sets the frequency of the monitoring cycles. There is a minimum of 60 and a maximum of 900 seconds (1 - 15 minutes).
  6. "Remove button" will remove it's associated item (i.e. for Attribute/Value entris, will delete the gruop of two)
  7. "Help button" will open a new tab displaying this page.

Running it

Finding the attribute

Before starting to check for stuff, you need to know what to search for. In general there are two ways to gather information:

  • You ask for it; - If you can, I strongly recommend you to ask arround if someone already has the attribute - value entries for what you want to monitor.
  • You search it; - This is what we will do in this section.

    • There are two ways that you can gather the attribute you need and it's value.

    Browser Dev tools
    1. Go to the page you want to monitor.
    2. Press F12 key on the keyboard to open the Development Tool in your browser.
    3. Click on selection tool in the Development tool.
    4. Select the object (element) you want to find on the page you want to monitor.
    5. In Delevopment Tool, check the attributes for the object you selected and try to find unique items.
    6. In Development Tool, press ctrl+f to make the find bar appear.
    7. Search the attribute you think is unique and check if there are more objects (elements) on that page.
    8. If the found element(s) is the one that you want, add them to the extension Attribute and Value fields.
    9. If not, try other combinations and other types of attibutes
    Selector tool (alpha state!)
    1. Go to the page you want to monitor.
    2. Press ctrl+shift+E keys on the keyboard to start the selector tool from the EM extension. If you hover on different elements of the website, a red outline will appear.
    3. When you found the element you want, click on it.
    4. Select the object (element) you want to find on the page you want to monitor.
    5. In Delevopment Tool, check the attributes for the object you selected and try to find unique items.
    6. In Development Tool, press ctrl+f to make the find bar appear.
    7. Search the attribute you think is unique and check if there are more objects (elements) on that page.
    8. If the found element(s) is the one that you want, add them to the extension Attribute and Value fields.
    9. If not, try other combinations and other types of attibutes

    Do it

    Important note: If you don't know how to get the attibute and it's value for the next section, please check Finding the element section before going forward.

    In order to set up a monitor you have to:

    1. Open a Tab with the desiered page (or if you already have it open, make the tab active);
    2. Open the extension popup;
    3. Click on Add/Remove current URL button;
    4. Add the attribute of the element you want to monitor in Attribute field;
    5. Add the value of the attrubute you want to monitor in Value field;
    6. CLick on Add button
